
# Utility Bill Management System (Client Side)

A MERN-stack based web application to manage monthly utility bills (Electricity, Gas, Water, Internet). Users can securely log in, pay bills, view detailed bill info, and download their paid bill history as a PDF. Built with **React, Vite, TailwindCSS, and Firebase Authentication**.

- **Bill Management**: Add, pay, and view bills (current month only for payments).
- **My Paid Bills**: Personalized table view of bills paid by the logged-in user.
- **Update/Delete Bills**: Update or delete paid bills via modal forms.
- **PDF Report**: Download a PDF report of all paid bills for the current user.
- **Responsive UI**: Mobile, tablet, and desktop friendly.
- **Dynamic Routing**: Single-page application using React Router.
- **Notifications**: Toast messages for all CRUD actions.
